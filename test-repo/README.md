# CodeSentinel Test Repository

A real Flask application with **intentional security vulnerabilities** for testing the CodeSentinel security scanning platform.

## ⚠️ DO NOT deploy this in production.

## How to use with CodeSentinel

1. **Push to your GitHub account:**
```bash
git init
git add .
git commit -m "Initial commit — CodeSentinel test application"
git remote add origin https://github.com/YOUR_USERNAME/codesentinel-test-app.git
git push -u origin main
```

2. **Install the CodeSentinel GitHub App** on this repository.

3. **Connect in CodeSentinel** → Repositories → Connect GitHub.

4. **Create a pull request** to trigger a full scan:
```bash
git checkout -b feature/test-scan
echo "# Minor change to trigger scan" >> README.md
git add . && git commit -m "chore: test PR to trigger CodeSentinel scan"
git push origin feature/test-scan
```

5. Then open a PR from `feature/test-scan` → `main` on GitHub.

6. **Watch the pipeline** in CodeSentinel → Scans.
